Code source de mon projet de recherche sur les réseaux de neurones à décharges
Lechekhab Samir, Lecs2813

Code 1:
	1_HelloWorldLIF.py
	Ce code permet de visualiser le potentiel
	de la membrane ainsi que l'émission des impulsion
Code 2 :
	2_Sinus_input_LIF.py
	Ce code permet d’analyser le comportement d’un
	Neurone de type LIF avec en entrée un courant sinusoïdal
Code 3 :
	3_music_input.py 
	Ce code montre comment convertir un fichier audio en un courant d’entrée 
	pour stimuler un neurone.
Code 4 :
	4_Rate_coding.py
	Ce code permet de visualiser la conversion analogique vers
	impulsions en incrémentant graduellement le courant d’entrée
Code 5 :
	5_SimpleSynapses.py
	Ce code est une introduction aux synapses, un premier neurone est stimulé 
	avec un courant continu et est connecté à trois autres neurones via des synapses
	de poids différents. 
Code 6 :
	6_LargeSynapsesConnectivity.py
	Ce code introduit une fonction permettant de représenter les liens synaptiques 
	entre plusieurs neurones et d’indiquer le poids de ce lien en fonction de la couleur.
Code 7 :	
	7_BasicSTDP.py
	Ce code donne un aperçu du fonctionnement de STDP
Code 8 : 
	8_Spectrograms.py
	Ce code permet de générer des spectrogrammes 
Code 9 :
	9_STDPLearning.py
	C'est le réseau de neurone qui aurai dû servir à apprendre à reconnaitre les notes de musique
	En regroupant tous les principes vus précédemment.
